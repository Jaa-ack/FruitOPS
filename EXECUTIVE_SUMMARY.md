## 📞 技術支援

### 常見問題

**Q: 如何修改水果品項或月份？**  
A: 編輯 `types/fruitCycle.ts` 中的 FRUIT_CYCLES 或 MONTH_INFO 常數

**Q: 行事曆為什麼沒有顯示？**  
A: 確保 Dashboard 中的「查看行事曆」按鈕被點擊，或檢查瀏覽器控制台

**Q: 如何在其他頁面使用季節數據？**  
A: 導入 `getAvailableFruitsByMonth()` 函數：
```typescript
import { getAvailableFruitsByMonth } from '../types/fruitCycle';
const fruits = getAvailableFruitsByMonth(currentMonth);
```

**Q: 系統是否支持動態定價？**  
A: 目前不支持，但 `SEASONAL_LOGIC_INTEGRATION.md` 中有實施框架
